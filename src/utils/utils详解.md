# Hash类
作用：Hash 类提供哈希函数，用于将任意数据映射为固定大小的整数。
整数哈希 (HashInt32 / HashInt64)    
:把输入的数字彻底打乱，让连续的数字（1,2,3…）产生看起来毫无规律的哈希值。
用于处理整数键或生成随机数序列。

MurmurHash3 (MurmurHash3 函数)
:利用乘法和位旋转这两种CPU极快的操作，在极短时间内实现高质量的随机混合。

FNV-1a (FNV1aHash 函数)
代码极简，没有循环移位，适合短数据或嵌入式环境。性能不错，但长数据下质量不如MurmurHash3

ComputeHash(const void* data, size_t n)
处理任意内存块（void*）的哈希计算。HashSlice 内部就是调用它。

ComputeHashWith(Algorithm algo, ...)
这是用于“切换”和“对比”的高级接口。
作用：让你可以运行时选择不同的哈希算法。

seed 参数（防御性设计）
这是为了保证“可控的随机性”。

作用：相同的输入，搭配不同的种子，会产生不同的哈希值。默认种子 0x9747b28c 是一个任意选择的魔法数。
